<h1 align="center"><PROJECT_NAME><br>by <TEAM_NAME><br><br>---<br><br>
### 🚀 Overview<br><PROJECT_NAME> is a Discord‑based prediction‑market arcade for live IPL matches.  <br>Users wager virtual points on simple yes/no questions (“Will Team X score ≥ Y runs in the next over?”) by buying & selling shares.  <br>All data (scores, win probabilities) is fetched live, and market prices adjust dynamically based on both team win‑% and supply & demand.<br><br>---<br><br>
### 🎯 Key Features<br>- **Auto‑Generated Markets**  <br>  – Bot creates new yes/no questions at configurable intervals  <br>  – Example: “Will the team score ≥ 48 runs by 5.0 overs?”  <br>- **Real‑Time Data**  <br>  – Ball‑by‑ball scores from a Cricket API  <br>  – Instant win‑probability model updates  <br>- **Virtual Points Economy**  <br>  – Each user starts with 20 points  <br>  – Correct prediction → stake returned + profit  <br>  – Wrong prediction → stake lost  <br>- **Dynamic Pricing**  <br>  – Prices move with live win‑odds and trade volume  <br>  – Heavy YES demand → YES price ↑, NO price ↓  <br>- **Tiered Ranking System**  <br>  – Codeforces‑style tiers: Newbie → Pupil → Specialist → Expert → Candidate Master → …  <br>  – Track total correct predictions; sync your Discord role with `!check_rank`  <br>- **Full Discord Integration**  <br>  – Admin commands: `!create_question`, `!resolve`  <br>  – User commands: `!list_questions`, `!market`, `!buy`, `!sell`, `!balance`, `!check_rank`<br><br>---<br><br>
### 🛠️ Tech Stack<br>- **Language:** Python 3.x  <br>- **Discord Library:** discord.py  <br>- **Data Fetching:** Cricket API (ball‑by‑ball + win‑prob)  <br>- **Persistence:** CSV logs for trades & balances  <br>- **Deployment:** Docker on AWS / VM<br><br>---<br><br>
### ⚙️ Prerequisites<br>- Python 3.8+  <br>- A Discord bot token  <br>- Cricket API key  <br><br>---<br><br>
### 🔧 Installation & Setup<br>1. **Clone the repository**  <br>2. **Create & activate a virtual environment**  <br>3. **Install dependencies**  <br>4. **Configure environment variables**  <br>- Copy `.env.example` to `.env`  <br>- Fill in:<br>  ```<br>  DISCORD_TOKEN=your_bot_token<br>  CRICKET_API_KEY=your_api_key<br>  ```<br>5. **Start the bot**  <br><br>---<br><br>
### 📖 Usage & Commands<br><br>| Command                       | Description                                   |<br>|-------------------------------|-----------------------------------------------|<br>| `!help`                       | Show help & full command list                 |<br>| `!list_questions`             | List all open prediction markets              |<br>| `!market <id>`                | Show details & current prices for a market    |<br>| `!buy <id> <opt> <shares>`    | Buy shares (deducts points)                   |<br>| `!sell <id> <opt> <shares>`   | Sell shares (before resolution)               |<br>| `!balance`                    | Display your balance, holdings & stats        |<br>| `!check_rank`                 | Show or force‑update your tier/rating         |<br>| `!create_question` *(admin)*  | Create a new market                           |<br>| `!resolve` *(admin)*          | Resolve a market & distribute payouts         |<br><br>---<br><br>
